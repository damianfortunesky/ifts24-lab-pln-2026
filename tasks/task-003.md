# Task 003 — Implementar procesamiento NLP y métricas analíticas

## Objetivo
Definir prompts para transformar texto en señales analíticas útiles con spaCy y NLTK.

## Prompt sugerido
"Actuá como especialista en PLN aplicado. Diseñá el módulo de procesamiento de texto para una app de análisis de agenda mediática en Python.

Debe incluir:
1) limpieza y normalización,
2) tokenización, lematización y POS tagging con spaCy,
3) extracción de entidades (personas, lugares, organizaciones),
4) frecuencias de términos/lemmas,
5) tópicos o agrupaciones simples (si no hay modelado avanzado, usar heurísticas interpretables),
6) métricas comparativas por medio y por fecha,
7) formato de salida para visualización (tablas y estructuras listas para gráficos).

Sumá recomendaciones para español (modelo spaCy sugerido, limitaciones y fallback).
"

## Criterio de aceptación
- El módulo devuelve resultados interpretables y comparables.
- Se explicitan límites metodológicos.
